About this Website

This is a preview of the Programming Club’s redesigned website.
Designed by Kentaro Watts.
Implemented by Joshua Wierenga and Kentaro Watts.

Built using Pandoc’s Markdown to HTML converter as a static site generator with images prepared with ImageMagick.
The website is hosted on Github Pages, with the original source stored on Github as well.
This website relies on Bootstrap 5, licensed under MIT and various icons from Bootstrap Icons, also licensed under MIT.
Special thanks to Nathaniel for their css only dark mode toggle code.
Using this AI preventing robots.txt(hopefully it works) by ai.robots.txt on Github under MIT.
