    • Minecraft Server

    How to join:

    1. Download Minecraft (Bedrock edition) on your Windows 10, Android, iPhone/iPad, Xbox, PlayStation, Switch, or any other device that can run Minecraft Bedrock edition
    2. Join the whitelist
      • You will need to be a member or sign-up as one
      • Get a ticket to agree to the servers rules join the servers whitelist
      • Contact the committee through our Discord server to get them to update the whitelist
    3. Connect to the server
      • Open minecraft and click play -> servers
      • Scroll down the list of servers and click add server
      • Enter these values
        • Server Name: Programming Club
        • Server Address: mc.programmingclub.com.au
        • Port: 19136
    4. Click "Save" and then "Join Server"
    5. Enjoy!
